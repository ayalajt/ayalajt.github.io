Tip Tool

Input Worker Information

Individually developed a website used by a local restaurant to calculate the tip payout for each worker per day.

Features:

  • Calculates the tip payout for each worker depending on their position and hours
  • Saves these daily payouts on a server so they can be accessed later
  • Previous payouts can be added together to view each worker's total within a certain date range
  • Tip payouts can be saved as a PDF
  • An extra tab, for managers only, allows them to input the total cash made and combine it with the tip payout to view the cash total for the day
  • These cash reports are also stored on a server so they can be accessed whenever necessary

Created in HTML, Javascript, deployed via Heroku, and information is stored via a cloud JSON API

Word Sense Disambiguation

Developed a program that is a Decision List classifier for disambiguating the word "line" given a training and testing file

Features:

  • Accepts a training and a testing file that each have different instances of the word "line"
  • Parses through the training file to gather information for each instance of the word "line", creating features to increase the accuracy of detecting what form of the word "line" is in
  • Uses the information gathered from the training file on the testing file
  • Compares the testing file results with a key, and outputs the results in a matrix with an overall accuracy score

Developed in Python
